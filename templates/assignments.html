Assignments!

My favorite assignment was the midterm project.

I enjoyed this project because we got to edit images. It was really rewarding
to see the images flip or become black and white and to know the actual coding
that goes into doing these types of edits. I have edited photos on my phone
before but never knew how it all works. I also thought it was interesting
how the images were just text files with numbers and that we could apply
methods that we learned to use on lists to images. It also just felt really
rewarding to be able to go from nothing (no skeleton code) to an entire
functioning code, and then to see it working successfully on the images.
